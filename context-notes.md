# Context Notes — 기찰반점 가맹본부 홈페이지

## 프로젝트 본질 (가장 중요)
이 사이트는 **가맹본부(바른미식)의 가맹점 모집용 홈페이지**다. 매장 손님용이 아니라
**예비 창업자(가맹희망자) 유치**가 1순위 목표. 따라서:
- 메뉴/음식 사진은 "맛있어 보이게"가 아니라 **"이 브랜드가 돈이 된다는 근거"**로 배치.
- 핵심 동선: 브랜드 강점 → 창업 비용/수익 → 가맹 절차 → 가맹문의(전환).
- 모든 CTA의 종착점은 "가맹문의" (전화 1600-5392 / 문의 폼).

## 기술 결정
- React 18 + Vite + TypeScript. 외부 UI 라이브러리 없이 순수 CSS(디자인 토큰).
- 지도: leaflet (npm). 인스타: 공식 oEmbed 스크립트(blockquote) — URL 3개 대기중.
- 배경영상: ffmpeg로 42초~ 약 10초 컷 + 1080p/web mp4 압축 → public/media.
- 디자인 톤: 중식 레드(#C0392B 계열) + 골드 포인트 + 다크 베이스, 모던/프리미엄.
